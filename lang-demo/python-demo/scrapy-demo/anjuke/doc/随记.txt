开始
    处理反爬
        如果请求被重定向
            1 - 下载器中间件 process_request 识别重定向 url 
            2 - 如果 url 是用于反爬的，则使用 Selenium 打开该 url 页面。
            3 - 用户在页面完成反爬验证
            4 - 使用重定向前的 url 构造请求，并返回。
        如果响应包含验证
            1 - 下载器中间件 process_response 识别反爬验证页面
            2 - 使用 Selenium 打开反爬验证页面
            3 - 用户在页面完成反爬验证
            4 - 使用原来的 url 构造请求，并返回。
