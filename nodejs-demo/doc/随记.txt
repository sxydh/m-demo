        npm
            指定镜像源
                npm install <package_name> --registry=https://registry.npm.taobao.org
                npm config set registry https://registry.npm.taobao.org
                    验证
                        npm config get registry
        yarn
            配置代理

                yarn config set https-proxy http://127.0.0.1:10808
