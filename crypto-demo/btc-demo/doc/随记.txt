概述
    比特币是虚拟货币
术语
    交易记录
        概述
            交易记录是用户之间交易币转移时形成的流水
        组成
            ID
                交易记录的唯一标识
            输入
                概述
                    交易记录的交易币来源，即交易前的交易币地址。
            输出
                概述
                    交易记录的交易币去向，即交易后的交易币地址。
            大小
                交易记录的数据大小
            交易费
                交易记录被矿工处理时，需要给矿工支付的费用。
            ...
    区块
        概述
            区块是多条交易记录的组织结构，即多条交易记录可以放在一起，形成一个区块。
            区块的大小固定为 1 兆
        组成
            区块头
                随机数
                    一个区块有一个随机数
                    随机数的哈希值需要满足特定条件
                    创建一个区块需要先创建一个有效的随机数，一般采用穷举算法，需要耗费大量算力。寻找随机数的过程称为挖矿，对找到有效随机数的用户会给予交易币奖励。
                上一个区块的哈希值
                    上一个区块的哈希值可以将所有区块连接起来，形成区块链。
            交易列表
                交易记录列表
            ...
    区块链
        概述
            区块链由所有的区块连接而成，连接的依据是区块头中包含的上一个区块的哈希值。
