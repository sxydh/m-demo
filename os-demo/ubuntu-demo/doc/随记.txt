常用命令
    进程
        列表
            ps aux 
                a ：包括所有进程。
                u ：以用户友好的格式显示信息。
                x ：包括没有控制终端的进程。
    软件
        列表
            apt list --installed 
        搜索
            apt search <package_key_name> 
                使用关键字搜索可安装的包
        卸载
            apt remove <package_name> 
        清理
            apt purge <package_name> 
    网络
        查看
            lsof -i:8080
                查看端口占用
    存储
        概述
            组成关系：物理介质 => 物理卷 PV => 卷组 VG => 逻辑卷 LV 。
        查看
            df -h
                查看磁盘空间使用情况
            sudo pvdisplay
                查看物理卷信息
        修改
            sudo lvextend -l +100%FREE /dev/mapper/ubuntu--vg-ubuntu--lv 
                lvextend ：扩展逻辑卷。
                -l ：指定逻辑卷大小，单位是百分比。
                +100%FREE ：将逻辑卷扩展到物理卷剩余空间的百分百。
                /dev/mapper/ubuntu--vg-ubuntu--lv ：逻辑卷路径。

                注意，扩展逻辑卷之后，还需要 resize2fs 扩展文件系统大小。

            sudo resize2fs /dev/mapper/ubuntu--vg-ubuntu--lv
                resize2fs ：调整文件系统大小。
                /dev/mapper/ubuntu--vg-ubuntu--lv ：逻辑卷路径。
    日志
        查看
            journalctl -u docker.service -n 20 -f
                查看 docker.service 服务的最新 20 条日志
                -u ：服务名称。
                -n ：日志条数。
                -f ：实时输出。
