                VMnet8
                    概述
                        VMnet8 使用 NAT （ Network Address Translation ，网络地址转换）技术，可以让虚拟机通过主机的 IP 地址访问外部网络。主机充当一个网关，将虚拟机的流量转发到外部网络。
                        NAT 与主机在一个子网中，但是 NAT 自己会创建另外一个子网，每台虚拟机的 IP 都在 NAT 创建的这个子网中。
