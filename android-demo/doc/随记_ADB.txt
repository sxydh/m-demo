开始
    devices
        设备列表
            adb devices
                    
    logcat
        日志列表
            adb -s [<device_id>] logcat *:E > logcat.txt
            
    push
        上传文件
            adb push <local_file> <remote_location>
            
    pull
        下载文件
            adb pull <android_path> <local_path>
            
    reboot
        重启设备
            adb reboot
            
    shell
        进入设备
            adb shell [<device_id>]
        
            开始
                am start -n com.android.settings/.Settings
                    am ： Activity Manager 。
                    -n ：组件名称。
                    
                    启动应用
                    
                am monitor
                    监控应用，可用于查看 Activity 的启动。
                
                getprop ro.product.cpu.abi
                    getprop ： Get Property 。
                    
                    查看处理器架构
                                
                pm list packages
                    pm ： Package Manager 。
                        
                    查看应用包名列表
                    
                dumpsys activity top
                    查看当前界面 Activity ，包含栈结构等信息。
                    
                uiautomator dump /sdcard/dump.xml
                    保存当前界面内容到指定 xml 文件
